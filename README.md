# An application for searching for books in the Google Books API

## Stack
* Vite, bundler
* React, library for building UI
* Tailwind, utility-first css framework
* Redux, for state-management
* Cypress, for CI tests

## Install dependencies

```shell
yarn install
```

## Development
```shell
yarn dev
```

## Building
```shell
yarn build
```

## Preview

```shell
yarn preview 
```
